# Εκτέλεση αυτού του δείγματος

## -1- Εγκαταστήστε τις εξαρτήσεις και δημιουργήστε το έργο

```bash
cargo build
```

## -2- Εκτελέστε το δείγμα

```bash
cargo run
```
