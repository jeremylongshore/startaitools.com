# Εκτέλεση δείγματος

## Εγκατάσταση εξαρτήσεων

```bash
npm install
```

## Δημιουργία

```bash
npm run build
```

## Εκκίνηση διακομιστή

```bash
npm start
```

αυτό θα ξεκινήσει τον διακομιστή

### Εκκίνηση πελάτη

Σε ένα ξεχωριστό τερματικό, εκτελέστε:

```bash
node run client
```
