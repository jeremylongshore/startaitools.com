# اجرای نمونه

## نصب وابستگی‌ها

```bash
npm install
```

## ساخت

```bash
npm run build
```

## راه‌اندازی سرور

```bash
npm start
```

این باید سرور را راه‌اندازی کند

### راه‌اندازی کلاینت

در یک ترمینال جداگانه اجرا کنید:

```bash
node run client
```
