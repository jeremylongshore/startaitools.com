# Spustenie ukážky

## Inštalácia závislostí

```bash
npm install
```

## Zostavenie

```bash
npm run build
```

## Spustenie servera

```bash
npm start
```

týmto by sa mal spustiť server

### Spustenie klienta

V samostatnom termináli spustite:

```bash
node run client
```
