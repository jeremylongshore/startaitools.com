# Pokreni primjer

## Instaliraj ovisnosti

```bash
npm install
```

## Izgradi

```bash
npm run build
```

## Pokreni poslužitelj

```bash
npm start
```

ovo bi trebalo pokrenuti poslužitelj

### Pokreni klijent

U zasebnom terminalu, pokreni:

```bash
node run client
```
