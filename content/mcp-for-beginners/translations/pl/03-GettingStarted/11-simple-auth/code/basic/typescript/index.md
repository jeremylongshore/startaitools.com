# Uruchom przykład

## Zainstaluj zależności

```bash
npm install
```

## Budowanie

```bash
npm run build
```

## Uruchom serwer

```bash
npm start
```

to powinno uruchomić serwer

### Uruchom klienta

W osobnym terminalu uruchom:

```bash
node run client
```
