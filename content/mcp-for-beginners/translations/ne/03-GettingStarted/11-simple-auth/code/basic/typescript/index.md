# नमूना चलाउनुहोस्

## निर्भरता स्थापना गर्नुहोस्

```bash
npm install
```

## निर्माण गर्नुहोस्

```bash
npm run build
```

## सर्भर सुरु गर्नुहोस्

```bash
npm start
```

यसले सर्भर सुरु गर्नुपर्छ

### क्लाइन्ट सुरु गर्नुहोस्

अर्को टर्मिनलमा, चलाउनुहोस्:

```bash
node run client
```
