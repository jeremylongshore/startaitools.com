# इस सैंपल को चलाना

आपको `uv` इंस्टॉल करने की सलाह दी जाती है लेकिन यह जरूरी नहीं है, देखें [instructions](https://docs.astral.sh/uv/#highlights)

## -1- डिपेंडेंसीज़ इंस्टॉल करें

```bash
npm install
```

## -3- सर्वर चलाएं

```bash
npm run build
```

## -4- क्लाइंट चलाएं

```sh
npm run client
```
