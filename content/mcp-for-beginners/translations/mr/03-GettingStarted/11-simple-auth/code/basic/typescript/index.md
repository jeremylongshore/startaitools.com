# नमुना चालवा

## आवश्यक गोष्टी स्थापित करा

```bash
npm install
```

## तयार करा

```bash
npm run build
```

## सर्व्हर सुरू करा

```bash
npm start
```

यामुळे सर्व्हर सुरू होईल

### क्लायंट सुरू करा

वेगळ्या टर्मिनलमध्ये, चालवा:

```bash
node run client
```
