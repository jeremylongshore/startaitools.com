# รันตัวอย่าง

## ติดตั้ง dependencies

```bash
npm install
```

## สร้างโปรเจกต์

```bash
npm run build
```

## เริ่มเซิร์ฟเวอร์

```bash
npm start
```

นี่จะเริ่มเซิร์ฟเวอร์

### เริ่มไคลเอนต์

ในเทอร์มินัลอีกหน้าต่างหนึ่ง ให้รัน:

```bash
node run client
```
