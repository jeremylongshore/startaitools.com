# รันตัวอย่าง

## ตั้งค่าสภาพแวดล้อมเสมือน

```sh
python -m venv venv
source ./venv/bin/activate
```

## ติดตั้ง dependencies

```sh
pip install "mcp[cli]"
```

## รันโค้ด

```sh
python client.py
```

คุณควรเห็นข้อความ:

```text
Available tools: ['add']
Result of add tool: meta=None content=[TextContent(type='text', text='8.0', annotations=None, meta=None)] structuredContent=None isError=False
```
