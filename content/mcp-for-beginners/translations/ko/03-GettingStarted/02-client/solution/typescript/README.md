# 이 샘플 실행하기

`uv` 설치를 권장하지만 필수는 아닙니다. 자세한 내용은 [instructions](https://docs.astral.sh/uv/#highlights)를 참고하세요.

## -1- 의존성 설치하기

```bash
npm install
```

## -3- 서버 실행하기

```bash
npm run build
```

## -4- 클라이언트 실행하기

```sh
npm run client
```

다음과 비슷한 결과가 보여야 합니다:

```text
Prompt:  {
  type: 'text',
  text: 'Please review this code:\n\nconsole.log("hello");'
}

Tool result:  { content: [ { type: 'text', text: '9' } ] }
```
