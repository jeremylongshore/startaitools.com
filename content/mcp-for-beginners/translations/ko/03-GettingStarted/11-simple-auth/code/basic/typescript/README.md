# 샘플 실행

## 의존성 설치

```bash
npm install
```

## 빌드

```bash
npm run build
```

## 서버 시작

```bash
npm start
```

이 명령은 서버를 시작합니다.

### 클라이언트 시작

다른 터미널에서 다음을 실행하세요:

```bash
node run client
```
