# Chạy mẫu

## Cài đặt các phụ thuộc

```bash
npm install
```

## Xây dựng

```bash
npm run build
```

## Khởi động máy chủ

```bash
npm start
```

Điều này sẽ khởi động máy chủ

### Khởi động client

Trong một terminal khác, chạy:

```bash
node run client
```
