# Chạy mẫu

## Thiết lập môi trường ảo

```sh
python -m venv venv
source ./venv/bin/activate
```

## Cài đặt các phụ thuộc

```sh
pip install "mcp[cli]"
```

## Chạy mã

```sh
python client.py
```

Bạn sẽ thấy văn bản:

