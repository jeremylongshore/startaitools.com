# הרצת הדוגמה הזו

מומלץ להתקין את `uv` אבל זה לא חובה, ראה [הוראות](https://docs.astral.sh/uv/#highlights)

## -1- התקן את התלויות

```bash
npm install
```

## -3- הפעל את השרת

```bash
npm run build
```

## -4- הפעל את הלקוח

```sh
npm run client
```
