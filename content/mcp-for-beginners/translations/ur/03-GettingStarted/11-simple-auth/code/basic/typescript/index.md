# نمونہ چلائیں

## ضروریات انسٹال کریں

```bash
npm install
```

## تعمیر کریں

```bash
npm run build
```

## سرور شروع کریں

```bash
npm start
```

یہ سرور کو شروع کر دینا چاہیے

### کلائنٹ شروع کریں

ایک الگ ٹرمینل میں، چلائیں:

```bash
node run client
```
