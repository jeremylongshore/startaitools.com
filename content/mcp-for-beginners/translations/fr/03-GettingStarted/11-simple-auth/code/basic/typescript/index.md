# Exécuter l'exemple

## Installer les dépendances

```bash
npm install
```

## Construire

```bash
npm run build
```

## Démarrer le serveur

```bash
npm start
```

Cela devrait démarrer le serveur

### Démarrer le client

Dans un terminal séparé, exécutez :

```bash
node run client
```
