# Стартиране на пример

## Инсталиране на зависимости

```bash
npm install
```

## Компилиране

```bash
npm run build
```

## Стартиране на сървър

```bash
npm start
```

това трябва да стартира сървъра

### Стартиране на клиент

В отделен терминал изпълнете:

```bash
node run client
```
