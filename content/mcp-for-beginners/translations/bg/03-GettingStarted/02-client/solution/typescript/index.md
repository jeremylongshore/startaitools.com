# Стартиране на този пример

Препоръчително е да инсталирате `uv`, но не е задължително, вижте [инструкциите](https://docs.astral.sh/uv/#highlights)

## -1- Инсталиране на зависимостите

```bash
npm install
```

## -3- Стартиране на сървъра

```bash
npm run build
```

## -4- Стартиране на клиента

```sh
npm run client
```

Трябва да видите резултат, подобен на:

```text
Prompt:  {
  type: 'text',
  text: 'Please review this code:\n\nconsole.log("hello");'
}
Resource template:  file
Tool result:  { content: [ { type: 'text', text: '9' } ] }
```
