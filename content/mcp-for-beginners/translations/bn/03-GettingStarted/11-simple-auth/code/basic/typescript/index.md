# নমুনা চালান

## নির্ভরশীলতা ইনস্টল করুন

```bash
npm install
```

## বিল্ড করুন

```bash
npm run build
```

## সার্ভার শুরু করুন

```bash
npm start
```

এটি সার্ভার শুরু করা উচিত

### ক্লায়েন্ট শুরু করুন

একটি আলাদা টার্মিনালে চালান:

```bash
node run client
```
