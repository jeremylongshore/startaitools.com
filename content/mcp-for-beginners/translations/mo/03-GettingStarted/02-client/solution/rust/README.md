# 執行此範例

> [!NOTE]
> 此 Rust 客戶端假設您已安裝 dotnet 8.0 SDK，以便在 [server](../../../../../../03-GettingStarted/02-client/solution/server) 目錄中執行範例 MCP 伺服器，該目錄與此專案為相鄰目錄。

## -1- 安裝依賴項並建置專案

```bash
cargo build
```

## -2- 執行範例

```bash
cargo run
```
