# 執行範例

## 安裝依賴項

```bash
npm install
```

## 建置

```bash
npm run build
```

## 啟動伺服器

```bash
npm start
```

這應該會啟動伺服器

### 啟動客戶端

在另一個終端中執行：

```bash
node run client
```
