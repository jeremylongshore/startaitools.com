# 執行此範例

## -1- 啟動可串流的 HTTP 伺服器

從此專案的根目錄開啟終端機，並執行以下指令：

```bash
cd ../calculator-httpserver
cargo run
```

保持此終端機開啟，因為它將運行客戶端連接的 HTTP 伺服器。接下來的步驟請開啟新的終端機。

## -2- 安裝依賴項並建置專案

```bash
cargo build
```

## -3- 執行範例

```bash
cargo run
```
