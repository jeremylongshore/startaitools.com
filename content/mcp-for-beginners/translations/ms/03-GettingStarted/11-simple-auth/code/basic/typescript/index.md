# Jalankan sampel

## Pasang kebergantungan

```bash
npm install
```

## Bina

```bash
npm run build
```

## Mulakan pelayan

```bash
npm start
```

ini sepatutnya memulakan pelayan

### Mulakan klien

Dalam terminal yang berasingan, jalankan:

```bash
node run client
```
