# تشغيل العينة

## تثبيت التبعيات

```bash
npm install
```

## البناء

```bash
npm run build
```

## بدء الخادم

```bash
npm start
```

يجب أن يؤدي هذا إلى بدء تشغيل الخادم

### بدء العميل

في نافذة طرفية منفصلة، قم بتشغيل:

```bash
node run client
```
