# Voorbeeld uitvoeren

## Afhankelijkheden installeren

```bash
npm install
```

## Bouwen

```bash
npm run build
```

## Server starten

```bash
npm start
```

dit zou de server moeten starten

### Client starten

Open in een aparte terminal:

```bash
node run client
```
