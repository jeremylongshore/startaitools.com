# Patakbuhin ang halimbawa

## I-install ang mga kinakailangan

```bash
npm install
```

## I-build

```bash
npm run build
```

## Simulan ang server

```bash
npm start
```

dapat magsimula ang server

### Simulan ang kliyente

Sa isang hiwalay na terminal, patakbuhin:

```bash
node run client
```
