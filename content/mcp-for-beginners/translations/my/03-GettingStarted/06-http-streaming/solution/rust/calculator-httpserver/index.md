# ဒီနမူနာကို အလုပ်လုပ်စေခြင်း

## -1- လိုအပ်သောအရာများကို ထည့်သွင်းပြီး ပရောဂျက်ကို တည်ဆောက်ပါ

```bash
cargo build
```

## -2- နမူနာကို အလုပ်လုပ်စေပါ

```bash
cargo run
```
