# နမူနာကို အလုပ်လုပ်စေပါ

## အခြေခံပတ်ဝန်းကျင်ကို စတင်ပြင်ဆင်ပါ

```sh
python -m venv venv
source ./venv/bin/activate
```

## လိုအပ်သောအရာများကို ထည့်သွင်းပါ

```sh
pip install "mcp[cli]"
```

## ကုဒ်ကို အလုပ်လုပ်စေပါ

```sh
python client.py
```
