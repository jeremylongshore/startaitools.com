# နမူနာကို အလုပ်လုပ်စေပါ

## လိုအပ်သောအရာများကို ထည့်သွင်းပါ

```bash
npm install
```

## တည်ဆောက်ပါ

```bash
npm run build
```

## Server ကို စတင်ပါ

```bash
npm start
```

ဒါက Server ကို စတင်ပေးမယ်

### Client ကို စတင်ပါ

အခြား Terminal မှာ အောက်ပါကို အလုပ်လုပ်စေပါ:

```bash
node run client
```
