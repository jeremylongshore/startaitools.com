# နမူနာကို အလုပ်လုပ်စေပါ

## ပတ်ဝန်းကျင်ကို ပြင်ဆင်ပါ

```sh
python -m venv
source ./venv/bin/activate
```

## လိုအပ်သောအရာများကို ထည့်သွင်းပါ

```sh
pip install PyJWT
```

## ကုဒ်ကို အလုပ်လုပ်စေပါ

```sh
python lab.py
```
