# Запуск примера

## Установка зависимостей

```bash
npm install
```

## Сборка

```bash
npm run build
```

## Запуск сервера

```bash
npm start
```

это должно запустить сервер

### Запуск клиента

В отдельном терминале выполните:

```bash
node run client
```
