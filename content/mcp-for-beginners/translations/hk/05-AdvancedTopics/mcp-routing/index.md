## MCP 中的抽樣與路由架構

抽樣是 Model Context Protocol (MCP) 的關鍵組成部分，能有效地處理和路由請求。它透過分析進來的請求，根據內容類型、使用者上下文和系統負載等多種條件，判斷最適合處理該請求的模型或服務。

抽樣與路由可以結合起來，打造一個強健的架構，優化資源利用率並確保高可用性。抽樣過程用於分類請求，而路由則將請求導向適當的模型或服務。
