# ਇਸ ਨਮੂਨੇ ਨੂੰ ਚਲਾਉਣਾ

> [!NOTE]
> ਇਹ Rust ਕਲਾਇੰਟ ਇਹ ਮੰਨਦਾ ਹੈ ਕਿ ਤੁਹਾਡੇ ਕੋਲ dotnet 8.0 SDK ਇੰਸਟਾਲ ਹੈ, ਤਾਂ ਜੋ ਤੁਸੀਂ [server](../../../../../../03-GettingStarted/02-client/solution/server) ਡਾਇਰੈਕਟਰੀ ਵਿੱਚ ਨਮੂਨਾ MCP ਸਰਵਰ ਚਲਾ ਸਕੋ, ਜੋ ਇਸ ਪ੍ਰੋਜੈਕਟ ਦੀ ਸਹਿਯੋਗੀ ਡਾਇਰੈਕਟਰੀ ਹੈ।

## -1- ਡਿਪੈਂਡੈਂਸੀਜ਼ ਇੰਸਟਾਲ ਕਰੋ ਅਤੇ ਪ੍ਰੋਜੈਕਟ ਬਣਾਓ

```bash
cargo build
```

## -2- ਨਮੂਨਾ ਚਲਾਓ

```bash
cargo run
```
