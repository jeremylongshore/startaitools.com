# ਇਸ ਨਮੂਨੇ ਨੂੰ ਚਲਾਉਣਾ

## -1- ਸਟ੍ਰੀਮ ਕਰਨ ਯੋਗ HTTP ਸਰਵਰ ਚਲਾਓ

ਇਸ ਪ੍ਰੋਜੈਕਟ ਦੇ ਰੂਟ ਤੋਂ ਇੱਕ ਟਰਮੀਨਲ ਖੋਲ੍ਹੋ ਅਤੇ ਹੇਠਾਂ ਦਿੱਤੇ ਕਮਾਂਡ ਚਲਾਓ:

```bash
cd ../calculator-httpserver
cargo run
```

ਇਸ ਟਰਮੀਨਲ ਨੂੰ ਖੁੱਲ੍ਹਾ ਰੱਖੋ ਕਿਉਂਕਿ ਇਹ HTTP ਸਰਵਰ ਚਲਾਏਗਾ ਜਿਸ ਨਾਲ ਕਲਾਇੰਟ ਜੁੜੇਗਾ। ਅਗਲੇ ਕਦਮਾਂ ਲਈ ਇੱਕ ਨਵਾਂ ਟਰਮੀਨਲ ਖੋਲ੍ਹੋ।

## -2- ਡਿਪੈਂਡੈਂਸੀਜ਼ ਇੰਸਟਾਲ ਕਰੋ ਅਤੇ ਪ੍ਰੋਜੈਕਟ ਬਣਾਓ

```bash
cargo build
```

## -3- ਨਮੂਨਾ ਚਲਾਓ

```bash
cargo run
```
