# ਨਮੂਨਾ ਚਲਾਓ

## Dependencies ਇੰਸਟਾਲ ਕਰੋ

```bash
npm install
```

## ਬਣਾਓ

```bash
npm run build
```

## ਸਰਵਰ ਸ਼ੁਰੂ ਕਰੋ

```bash
npm start
```

ਇਸ ਨਾਲ ਸਰਵਰ ਸ਼ੁਰੂ ਹੋਣਾ ਚਾਹੀਦਾ ਹੈ

### ਕਲਾਇੰਟ ਸ਼ੁਰੂ ਕਰੋ

ਇੱਕ ਵੱਖਰੇ ਟਰਮੀਨਲ ਵਿੱਚ, ਚਲਾਓ:

```bash
node run client
```
