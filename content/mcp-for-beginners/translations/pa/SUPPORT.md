# ਸਹਾਇਤਾ

## ਮੁੱਦੇ ਦਰਜ ਕਰਨ ਅਤੇ ਮਦਦ ਪ੍ਰਾਪਤ ਕਰਨ ਦਾ ਤਰੀਕਾ  

ਇਹ ਪ੍ਰੋਜੈਕਟ ਬੱਗਾਂ ਅਤੇ ਫੀਚਰ ਬੇਨਤੀਆਂ ਨੂੰ ਟਰੈਕ ਕਰਨ ਲਈ GitHub Issues ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ। ਕਿਰਪਾ ਕਰਕੇ ਨਵੇਂ ਮੁੱਦੇ ਦਰਜ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਮੌਜੂਦਾ ਮੁੱਦਿਆਂ ਨੂੰ ਖੋਜੋ ਤਾਂ ਜੋ ਡੁਪਲਿਕੇਟ ਤੋਂ ਬਚਿਆ ਜਾ ਸਕੇ। ਨਵੇਂ ਮੁੱਦਿਆਂ ਲਈ, ਆਪਣਾ ਬੱਗ ਜਾਂ ਫੀਚਰ ਬੇਨਤੀ ਨਵੇਂ Issue ਵਜੋਂ ਦਰਜ ਕਰੋ।

ਇਸ ਪ੍ਰੋਜੈਕਟ ਦੀ ਵਰਤੋਂ ਬਾਰੇ ਮਦਦ ਅਤੇ ਸਵਾਲਾਂ ਲਈ, ਕਿਰਪਾ ਕਰਕੇ:
- [Azure AI Foundry Discord](https://discord.com/invite/ByRwuEEgH4) ਨੂੰ ਚੈੱਕ ਕਰੋ ਜਿੱਥੇ ਮਾਹਿਰ ਅਤੇ ਹੋਰ ਡਿਵੈਲਪਰ ਇਕੱਠੇ ਹੁੰਦੇ ਹਨ
- ਪ੍ਰੋਟੋਕੋਲ-ਸੰਬੰਧੀ ਸਵਾਲਾਂ ਲਈ [Model Context Protocol Community](https://modelcontextprotocol.io/community/) ਦੇ ਸਰੋਤਾਂ ਦੀ ਵਰਤੋਂ ਕਰੋ

## Microsoft Support Policy  

Model Context Protocol (MCP) for Beginners ਪ੍ਰੋਜੈਕਟ ਲਈ ਸਹਾਇਤਾ ਉਪਰੋਕਤ ਸਰੋਤਾਂ ਤੱਕ ਸੀਮਿਤ ਹੈ।

**ਅਸਵੀਕਾਰੋਪਣ**:  
ਇਹ ਦਸਤਾਵੇਜ਼ AI ਅਨੁਵਾਦ ਸੇਵਾ [Co-op Translator](https://github.com/Azure/co-op-translator) ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਅਨੁਵਾਦ ਕੀਤਾ ਗਿਆ ਹੈ। ਜਦੋਂ ਕਿ ਅਸੀਂ ਸਹੀਅਤ ਲਈ ਕੋਸ਼ਿਸ਼ ਕਰਦੇ ਹਾਂ, ਕਿਰਪਾ ਕਰਕੇ ਧਿਆਨ ਰੱਖੋ ਕਿ ਸਵੈਚਾਲਿਤ ਅਨੁਵਾਦਾਂ ਵਿੱਚ ਗਲਤੀਆਂ ਜਾਂ ਅਸਮਰਥਤਾਵਾਂ ਹੋ ਸਕਦੀਆਂ ਹਨ। ਮੂਲ ਦਸਤਾਵੇਜ਼ ਆਪਣੀ ਮੂਲ ਭਾਸ਼ਾ ਵਿੱਚ ਪ੍ਰਮਾਣਿਕ ਸਰੋਤ ਮੰਨਿਆ ਜਾਣਾ ਚਾਹੀਦਾ ਹੈ। ਮਹੱਤਵਪੂਰਨ ਜਾਣਕਾਰੀ ਲਈ, ਪੇਸ਼ੇਵਰ ਮਨੁੱਖੀ ਅਨੁਵਾਦ ਦੀ ਸਿਫਾਰਸ਼ ਕੀਤੀ ਜਾਂਦੀ ਹੈ। ਇਸ ਅਨੁਵਾਦ ਦੀ ਵਰਤੋਂ ਤੋਂ ਉਤਪੰਨ ਕਿਸੇ ਵੀ ਗਲਤਫਹਮੀ ਜਾਂ ਗਲਤ ਵਿਆਖਿਆ ਲਈ ਅਸੀਂ ਜ਼ਿੰਮੇਵਾਰ ਨਹੀਂ ਹਾਂ।
