# Mintapéldák futtatása

## Függőségek telepítése

```bash
npm install
```

## Buildelés

```bash
npm run build
```

## Szerver indítása

```bash
npm start
```

Ez elindítja a szervert.

### Ügyfél indítása

Egy külön terminálban futtasd:

```bash
node run client
```
