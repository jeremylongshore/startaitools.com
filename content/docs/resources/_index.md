## How to Use Templates

1. Reference the template in your AI assistant:
   ```
   Use @ai-dev-tasks/create-prd.md to create a PRD for [feature]
   ```

2. The AI will:
   - Read the template
   - Ask clarifying questions
   - Generate the document
   - Save with proper naming
