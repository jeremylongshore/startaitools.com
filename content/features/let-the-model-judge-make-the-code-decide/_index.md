Daily posts used to die when a model session hung, timed out, or wrote half a file. The fix was not a smarter prompt. It was a hard seam: the model may judge and draft; bash verifies preconditions, commits, dual-publishes, and quarantines anything incomplete. If the produce step fails, tomorrow still runs.
